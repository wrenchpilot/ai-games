# Asteroids Game

A classic arcade-style Asteroids game built with JavaScript and HTML5 Canvas.

## How to Play

Navigate to the game URL and use the following controls:

- Arrow keys to move the ship
- Space bar to shoot

Try to shoot all the asteroids while avoiding collisions!
